# printing hello world 10 times

## outside a list

```python,linenos
for _ in range(10):
    print('hello world')
exit()
```

## inside an unordered list

- first time:

  ```python,linenos
  for _ in range(10):
      print('hello world')
  exit()
  ```

- second time:

  ```python,linenos
  for _ in range(10):
      print('hello world')
  exit()
  ```

## inside an ordered list

1. second time:

   ```python,linenos
   for _ in range(10):
       print('hello world')
   exit()
   ```

1. second time:

   ```python,linenos
   for _ in range(10):
       print('hello world')
   exit()
   ```
